# Ecommerce Database Operations

## 📌 Overview

This repository contains practical SQL development, PostgreSQL database administration, monitoring, optimization, and application support operations performed using the Brazilian E-Commerce Dataset.

The project demonstrates real-world database tasks including:

- SQL query development
- Data analysis
- Query optimization
- PostgreSQL monitoring
- Performance tuning
- Application support scenarios
- Basic DBA operations
- Troubleshooting and debugging

---

# 🛠 Technologies Used

- PostgreSQL
- pgAdmin
- SQL
- PostgreSQL System Views
- Query Optimization Techniques
