# Ecommerce Database Operations

## 📌 Overview

This repository contains practical PostgreSQL database development, advanced SQL programming, query optimization, monitoring, and database administration operations performed using the Brazilian E-Commerce Dataset.

The project demonstrates real-world database tasks including:

- SQL query development
- Advanced SQL operations
- Triggers, Functions & Views
- Query optimization
- PostgreSQL monitoring
- Performance tuning
- Application support scenarios
- Basic DBA operations
- Troubleshooting and debugging
